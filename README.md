# 🖼️ React Gallery App

A simple Gallery App built with React and Axios.  
It fetches and displays images from an API with a clean UI.

---

## 🚀 Features

- Fetch images using Axios
- Display image gallery
- Responsive design
- Fast and simple interface

---

## 🛠️ Tech Stack

- React.js
- Axios
- CSS
- Vite
